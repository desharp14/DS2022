\section{设计思路}
\begin{itemize}
\item 通过 double strtd 和 string dts 转换 double 类型的数据和 string 类型的数据；
\item check 函数判断输入是否合法；
\item insert 插入数与符号；
\item priority 判断运算符的优先级;
\item 最终 calculate 完整计算。
\end{itemize}
